# Econ Agents Plugin

Specialist agents for Bellman-DDSL / computational economics development.

## Agents

- **bellman-architect** — Architecture & planning (readonly)
- **ceo** — Executive orchestrator, coordinates other agents
- **econ-code-critic** — Strategic reviewer
- **econ-model-maker** — Implementation specialist
- **econ-tribe** — Economist readability reviewer (readonly)
