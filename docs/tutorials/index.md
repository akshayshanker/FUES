# Notebooks

Notebook walkthroughs of the algorithm and the benchmark models. They
complement the model pages in [Applications](../examples/index.md) and
the command-line material in [Running locally](../running-locally.md).

| Notebook | Focus |
|---|---|
| [EGM / FUES walkthrough](../notebooks/egm_fues_transparent.ipynb) | Raw EGM correspondence, upper-envelope logic, and the role of the jump test |
| [Retirement benchmarks](../notebooks/retirement_fues.ipynb) | Finite-horizon retirement model and timing comparison across methods |
| [Durables (Cobb–Douglas)](../notebooks/durables_fues.ipynb) | Continuous housing adjustment with FUES and competing methods |
| [Durables (separable)](../notebooks/durables_fues_separable.ipynb) | Separable-utility variant and its diagnostics |

The notebooks render statically. To execute them, clone the repo and run
Jupyter locally (see [Installation](../getting-started/installation.md)).
